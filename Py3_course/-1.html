4 正则表达式

  • 有些人在碰到问题时，就想：“我知道，我可以使用正则表达式！”现在，他们就有了两个问题。（by Jamie “jwz” Zawinski）
  • 正则表达式是一些由字符和特殊符号组成的字符串，它们描述了模式的重复或者表述多个字符，于是正则表达式能按照某种模式匹配一系列有相似特征的字符串
  • 最简单的正则表达式：foo->foo, Python->Python, abc123->abc123
  • 引入特殊符号来定义字符集、匹配子组和重复模式
  • 这时正则表达式匹配的是字符集而不是单个的字符串
import re
# match()函数试图从字符串起始部分对模式进行匹配，如果匹配成功，就返回一个匹配对象，如果匹配失败，则返回None
# "You"是模式，“Young Frankenstein”是源
result=re.match("You", "Young Frankenstein")
if result is not None:
    print(result.group() )
print (result)
You
<_sre.SRE_Match object; span=(0, 3), match='You'>
result=re.match("i", "Sheng Xin Bao Dian")
if result is not None:
    print(result.group())
# search()的工作方式与match()完全一致，不同之处在于search()可以在任意位置对给定的正则表达式模式搜索出第一次出现的匹配情况
# 如果搜索到一个成功匹配，就会返回一个匹配对象，否则返回None
result=re.search("foo", "seafood")
if result is not None:
    print(result.group())
foo
result=re.search("i.", "Sheng Xin Bao Dian")
if result is not None:
    print(result.group())
in
# findall()查询字符串中某个模式全部的出现情况，findall()总是返回一个列表
re.findall("car", "carry the bacardi to the car")
['car', 'car', 'car']
result=re.findall("i.", "Sheng Xin Bao Dian")
result
['in', 'ia']
re.findall("[a-zA-Z0-9]+", "Sheng Xin Bao Dian")
['Sheng', 'Xin', 'Bao', 'Dian']
re.findall("[a-zA-Z0-9]*", "Sheng Xin Bao Dian")
['Sheng', '', 'Xin', '', 'Bao', '', 'Dian', '']
re.findall("[a-zA-Z0-9]{1,}", "Sheng Xin Bao Dian")
['Sheng', 'Xin', 'Bao', 'Dian']
re.findall("[a-zA-Z0-9]{1}", "Sheng Xin Bao Dian")
['S', 'h', 'e', 'n', 'g', 'X', 'i', 'n', 'B', 'a', 'o', 'D', 'i', 'a', 'n']

# finditer()在匹配对象中迭代, re.I -- Ignorecase
s = "This and that"
print(re.findall(r'(th\w+) and (th\w+)', s, re.I))
print([g.groups() for g in re.finditer(r'(th\w+) and (th\w+)', s, re.I)])
[('This', 'that')]
[('This', 'that')]
